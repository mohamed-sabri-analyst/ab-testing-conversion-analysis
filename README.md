# A/B Testing Conversion Analysis Dashboard

## 📊 Project Overview

This project analyzes an A/B testing experiment to evaluate user behavior and conversion performance using Power BI and Python.

---

## 🎯 Objective

To determine which version (A or B) performs better in terms of conversion rate and business impact.

---

## 🛠️ Tools Used

* Power BI (Dashboard & Visualization)
* DAX (Measures & KPIs)
* Python (EDA & Statistical Analysis)
* Excel (Data Source)

---

## 📈 Key Metrics

* Conversion Rate (A vs B)
* Conversion Lift
* Average Revenue

---

## 🔍 Key Insights

* Version B achieved a **+3% increase in conversion rate**
* No significant change in average revenue

---

## 💡 Recommendation

Adopt Version B to improve conversion performance while maintaining stable revenue.
